Blender 2.59 MD5 Export script
===============================
Script by keless.
Development topic - http://www.katsbits.com/smforum/index.php?topic=167.0

Installation
============
For extensive installation and usage instruction read the following topic.
http://www.katsbits.com/smforum/index.php?topic=178.0

Export script need to be placed in the following location;

	scripts\addons

or it should be loaded through User Preferences

	File > User Preferences... > (Install Add-Ons...)

For Windows Vista/Win7 users Blender's scripts folder is typically located in;

	C:\Users\[profile]\AppData\Roaming\Blender Foundation\Blender2.5x\scripts\addons

Windows XP users should look in; 

	C:\Documents and Settings\[profile]\Application Data\Blender Foundation\Blender2.5x\scripts\addons

If correctly installed the "Quake Model 5 (.md5)" should appear in the "Export" list

IMPORTANT NOTE: for 2.58+/- the "AddOn" folder is now located in Blender Installation directory and NOT in "user" as above.

Usage
=====
Select both the mesh object and armature to be exported, preferably the mesh should be selected last so it's highlight brighter orange. From the "File" menu select "File >> Export >> Quake Model 5 (.md5)". To export objects actual size leave "Scale: 1.0000" as is.
